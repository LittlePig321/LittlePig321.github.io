Anki with code

Published 2019-05-15 Tools

I happened to see an article introducing Anki last week. I suddenly had curious. So, I just want to make good use of this tool. First, download the software, then familiarize with the most basic functions of the software. I feel very good. Later I was thinking, I want to combine "Aurora Word", "Hundred Words", "Anki", so I have the following video.

After a week of familiarity with the software, and learning. By writing code, download the resources, mainly including pictures, audio, video. In this case, I do not need to repeatedly call the interface. Because "Aurora word" requires a cookie to crawl the picture.

Anki's plugin is quite rich. I mainly use "WordQuery," "Awesome TTS".

note:

  1. If you have a video request, don't use Anki2.1. Instead, use Anki2.0.
  2. You can develop your own templates, but it dosen't support ES6. There are too many pits in it.
